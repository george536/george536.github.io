About This Project

  • Participated in the Google Solutions Incubator Program at McMaster University.
  • Designed a device to help associates and retail workers determine their readiness for shifts by taking their temperature and answering routine questions.
  • Developed both the front-end and back-end for the device, ensuring functionality of hardware components and accurate integration.
  • Established database connections using Python with MySQL and created a GUI using Python's Tkinter library, displayed on a Raspberry Pi 4.
  • The project received the **Highest Impact Award** for its innovative design.
