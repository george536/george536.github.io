About This Project

  • Developed a GIPPS car-following model simulation for autonomous vehicles using RC cars

  • Designed and tuned a PID controller for the RC cars to follow a pre-set path using camera feed

  • Implemented communication protocol for the interaction of the RC cars using MQTT messaging protocol

  • Simulated a Traffic manager that controls traffic flow for the RC cars crossing through intersections

  • Conducted simulations to test and evaluate the performance of the model
